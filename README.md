# Circuitos Contador con VHDL

Este repositorio contiene el diseño e implementación de un circuito contador desarrollado en **VHDL**. El objetivo del proyecto es aplicar conceptos de diseño digital mediante la descripción de hardware, permitiendo simular y validar el funcionamiento del circuito antes de su implementación en una FPGA.

* Diseño del contador utilizando VHDL.

* Proyecto orientado a prácticas de electrónica digital y arquitectura de computadores.
